# 🇮🇳 JobBoard - Modern Job Search Platform

A beautiful, responsive job board application focused on the Indian job market with global opportunities. Built with modern web technologies and featuring a sleek glass-morphism design.
![image](https://github.com/user-attachments/assets/d9f6b216-dfd6-4062-b3ba-abbee49f4532)


## ✨ Features

### 🎯 Core Functionality
- **Advanced Job Search** - Search by title, company, skills, and location
- **Smart Filtering** - Filter by job type, experience level, salary range, and more
- **Responsive Design** - Perfect experience on desktop, tablet, and mobile
- **Dark/Light Mode** - Toggle between themes for comfortable viewing
- **Real-time Search** - Instant results as you type

### 🏢 Job Management
- **Job Listings** - Browse thousands of opportunities across India and globally
- **Company Profiles** - Detailed company information with ratings and reviews
- **Application Tracking** - Track your job applications and their status
- **Job Alerts** - Get notified about new opportunities matching your criteria
- **Salary Comparison** - Compare salaries across different roles and locations

### 🌟 Indian Market Focus
- **Indian Companies** - TechMahindra, Flipkart, Zomato, Paytm, Razorpay, and more
- **Indian Cities** - Bangalore, Mumbai, Delhi NCR, Hyderabad, Pune, Chennai
- **Local Benefits** - Festival bonuses, provident fund, cab facilities, food allowances
- **Cultural Elements** - Indian emojis, local context, and cultural celebrations
- **INR Salary Ranges** - Salary information in Indian Rupees

### 🎨 Modern UI/UX
- **Glass Morphism Design** - Modern glass-like effects with backdrop blur
- **Smooth Animations** - Engaging micro-interactions and transitions
- **Responsive Navbar** - Mobile-first navigation with slide-in menu
- **Interactive Cards** - Hover effects and dynamic content
- **Gradient Themes** - Beautiful teal/cyan/blue color palette

## 🛠️ Tech Stack

- **Frontend**: React 18, Next.js 14, TypeScript
- **Styling**: Tailwind CSS, CSS Modules
- **Icons**: Lucide React
- **State Management**: React Hooks, Local Storage
- **Build Tool**: Next.js with Turbopack
- **Deployment**: Vercel (recommended)

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn or pnpm

### Installation

1. **Clone the repository**
   \`\`\`bash
   git clone https://github.com/abhiii9vvv/jobboard.git
   cd jobboard
   \`\`\`

2. **Install dependencies**
   \`\`\`bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   \`\`\`

3. **Run the development server**
   \`\`\`bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   \`\`\`

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

## 📱 Usage

### For Job Seekers
1. **Search Jobs** - Use the search bar to find opportunities
2. **Apply Filters** - Narrow down results using advanced filters
3. **Save Jobs** - Bookmark interesting positions for later
4. **Track Applications** - Monitor your application status
5. **Set Alerts** - Get notified about new matching jobs

### For Employers
1. **Post Jobs** - Add new job listings
2. **Manage Applications** - Review and respond to candidates
3. **Company Profile** - Showcase your company culture
4. **Analytics** - Track job posting performance
